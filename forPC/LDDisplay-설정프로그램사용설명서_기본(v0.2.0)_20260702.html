LDDisplay 시리얼 제어 GUI — 사용설명서

이 설명서는 보드 1대에 USB로 연결해 밝기 / 폰트 / 스크롤 속도를 변경하는 절차입니다. 마스터·슬레이브 보드는 각각 따로 연결해 개별 설정합니다 (값은 보드별로 보관되며, 환경에 맞춰 따로 조정하는 것이 정상).

1. 화면 구성 한눈에

  1. 상단 — 프로그램 이름과 보드 펌웨어 버전 (ver.0.1.x)

    왼쪽: 프로그램 이름·PC 프로그램 버전 · 오른쪽: 연결된 보드의 펌웨어 버전(연결 시 자동 표시).

  2. 연결 — 포트 / 속도 / 연결 버튼 / 연결 상태 램프
  3. 기본 탭 — 밝기, 폰트, 스크롤 (한 줄)
  4. 색상 탭 — 상태별 화면 색상 (R/G/B) 조정 v0.2.0 신규
  5. RX (수신) — 보드에서 보낸 응답이 실시간으로 표시
  6. TX (송신) — 직접 명령을 타이핑해 보낼 수 있는 입력칸

2. 연결

2-1. 절차

  1. USB 케이블로 보드와 PC 연결.
  2. 포트 드롭다운에서 보드 COM 포트를 선택. (목록이 비어있으면 옆의 ⟳ 버튼)
  3. 속도는 기본값 115200 그대로 둡니다.
  4. 연결 버튼 클릭. 램프가 확인중(주황) → 연결됨(녹색) 으로 바뀌면 정상.
  5. 녹색이 되면 아래 컨트롤이 모두 활성화됩니다.

2-2. 연결 상태 램프

표시의미대응
미연결연결 전 상태포트 선택 후 [연결] 클릭
확인중포트는 열림. 보드 응답 대기 중1.5초 정도 기다림
연결됨보드 응답 정상. 컨트롤 사용 가능—
오류연결 실패 또는 통신 끊김케이블/포트 점검 후 재시도

응답 없음(오류)이 뜨면 — ① 케이블 분리·재연결 → ② ⟳ 클릭 후 포트 재선택 → ③ 다른 시리얼 모니터(예: Arduino IDE)가 같은 포트를 점유하고 있지 않은지 확인하세요.

3. 기본 탭 — 설정 항목

3-1. 밝기 (bright)

슬라이더로 값을 잡은 뒤 우측 [적용] 버튼을 눌러야 보드에 반영됩니다. (드래그만으로는 적용 안 됨)

항목값
일반 범위(기본 잠금)0 ~ 150
잠금 해제 시 범위0 ~ 255
출고 기본값96

밝기 잠금 해제 (151 이상으로 올리고 싶을 때)

  1. 밝기 슬라이더 오른쪽의 ≥151 체크박스를 켭니다.
  2. 슬라이더 상한이 자동으로 255까지 확장됩니다.
  3. 원하는 값으로 슬라이더 이동 → [적용] 클릭.
  4. 다시 잠그려면 체크박스를 끄세요. 슬라이더 상한이 150으로 복귀하며, 그보다 높은 값이면 150으로 자동 하향됩니다.

잠금은 발열·눈부심·소비전력 보호 목적입니다. 151 이상은 단시간/특수 환경에서만 사용하시기 바랍니다.

3-2. 폰트 (font)

한글 글리프의 굵기 선택. 두 버튼 중 하나가 항상 눌린(파란색) 상태로 표시됩니다.

  • Bold — 두꺼운 글자 (출고 기본)
  • Normal — 일반 굵기

버튼을 누르면 즉시 보드에 반영됩니다.

3-3. 스크롤 속도 (scroll)

흘러가는 텍스트(예: 운전중 / 시험중 / 정지중)의 속도. ▲ / ▼ 버튼으로 한 단계씩 조정.

값의미
1가장 느림
3중간 (출고 기본)
5가장 빠름

한계값(1 또는 5)에 도달하면 해당 방향 버튼이 비활성화됩니다.

4. 색상 탭 — 상태별 화면 색상 v0.2.0 신규

화면에 표시되는 4가지 상태의 색을 원하는 값으로 바꿀 수 있습니다. 각 상태에 딸린 "수동" 표시(수동활선·수동휴전·조작시험)의 색도 함께 바뀝니다.

4-1. 상태 4행이 의미하는 것

행 이름화면에 뜨는 문자출고 기본값 R, G, B
운전중 · 수동활선"운전중"(흐름) / "수동활선"(정지)255, 0, 0
정지중 · 수동휴전"정지중"(흐름) / "수동휴전"(정지)0, 255, 80
정지중(ES) · 조작시험"시험중"(흐름) / "조작시험"(정지)255, 255, 0
비정상"비정상"(흐름)64, 64, 64

"수동표시" 접두 화면은 뒤에 이어지는 상태(수동활선/수동휴전/조작시험)의 색과 자동으로 맞춰 표시됩니다.

4-2. 색 바꾸는 3단계

  1. 바꾸고 싶은 행의 R / G / B 칸에 0~255 값을 타이핑. 오른쪽 색상 상자가 즉시 그 색으로 바뀝니다 (미리보기).
  2. 같은 행의 [설정] 버튼 클릭 → 상태 라벨이 "저장 대기"로 바뀌면 그 값이 확정됩니다. (아직 보드에는 안 보냄)
  3. 맨 아래 [전체 저장] 버튼 클릭 → 이때 비로소 실제 LED에 반영되고 다음 부팅에도 유지됩니다.

[설정]만 눌러도 화면은 그대로입니다. 반드시 [전체 저장]까지 눌러야 LED에 반영되고 전원을 껐다 켜도 유지됩니다. 저장하지 않고 프로그램을 종료하면 편집 내용은 사라집니다.

4-3. 하단 버튼

버튼의미
[현재값 읽기]보드에 저장돼 있는 지금의 색상 4개를 읽어와 입력칸/색상 상자에 채웁니다. (연결 직후 자동으로 한 번 동기화되므로 평소엔 필요 없음)
[기본값 복원]입력칸을 출고 기본값(위 표)으로 채웁니다. 저장은 별도 — [전체 저장]을 눌러야 실제 반영됩니다.
[전체 저장]4개 행의 값을 한꺼번에 보드로 보냅니다 → LED + 비휘발성 메모리에 저장.

4-4. 상태 라벨이 뜻하는 것

라벨의미
저장됨입력값 = 보드에 저장된 값 (반영 완료)
편집중R/G/B를 바꾸는 중 (아직 [설정] 안 누름)
저장 대기[설정]으로 확정했지만 아직 [전체 저장] 전 → 보드에 아직 안 갔음
0~255 정수 필요입력한 값이 숫자가 아니거나 범위 밖

5. 변경 사항이 저장되는 방식

  • 밝기·폰트·스크롤·상태별 색상 값은 모두 보드의 비휘발성 메모리에 저장되므로 전원을 껐다 켜도 그대로 유지됩니다.
  • 각 보드는 자기 값만 보관합니다. 마스터를 바꿔도 슬레이브 값은 바뀌지 않으니, 두 보드를 같은 설정으로 맞추고 싶다면 USB를 옮겨서 각각 설정해 주세요. (색상도 동일 — 마스터/슬레이브를 같은 색으로 맞추려면 두 대 모두 저장 필요)
  • 설치 환경(주변 조명, 시야 거리)에 따라 적정 밝기가 다르므로, 보드별로 따로 잡는 것이 정상입니다.

6. RX / TX 창 (선택 사용)

6-1. RX (수신)

보드가 보낸 응답이 시각과 함께 누적 표시됩니다.

  • OK: 로 시작하면 명령이 정상 적용됨.
  • invalid command가 보이면 명령 철자/값 범위를 다시 확인.
  • 가득 차면 우측 하단 [RX 지우기]로 비울 수 있습니다.

6-2. TX (송신)

버튼 대신 명령을 직접 타이핑해서 보낼 수 있습니다. Enter로 전송.

status              현재 설정 모두 출력
help                명령 목록
bright 120          밝기를 120으로
font bold           폰트 굵게
scroll 2            스크롤 속도 2
getcolor            상태별 색상 조회
setcolor run 255 0 0  운전중 색을 R=255,G=0,B=0으로

7. 명령 한눈에 보기

명령범위 / 값의미
bright N0 ~ 150 (잠금 해제 시 0 ~ 255)밝기
font bold / font normal—폰트 굵기
scroll N1 ~ 5스크롤 속도
getcolor—상태별 색상 4개 조회 v0.2.0
setcolor tag R G Btag = run/stop/es/abn
R,G,B = 0~255		상태별 색상 지정 후 저장 v0.2.0
status—현재 설정 조회 (색상 포함)
help—명령 도움말

setcolor의 tag 의미 — run: 운전중·수동활선 / stop: 정지중·수동휴전 / es: 정지중(ES)·조작시험 / abn: 비정상.

8. 자주 묻는 문제

램프가 계속 주황(확인중)에서 멈춥니다

1.5초 안에 응답이 없으면 자동으로 적색(오류)으로 전환됩니다. 이때:

  • 케이블·포트가 보드와 실제로 연결돼 있는지 확인
  • 다른 프로그램(Arduino IDE 시리얼 모니터 등)이 동일 포트를 잡고 있지 않은지 확인
  • 속도(baud rate)가 115200 인지 확인

슬라이더는 움직이는데 보드가 변하지 않습니다

밝기는 [적용] 버튼을 눌러야 전송됩니다. 폰트/스크롤은 버튼 누름 즉시 전송됩니다.

밝기를 더 올리고 싶은데 슬라이더가 150에서 더 안 올라갑니다

슬라이더 옆의 ≥151 체크박스를 켜면 255까지 확장됩니다. (3-1 참고)

전원을 다시 켜면 설정이 초기화되나요?

아니요. 변경 즉시 보드의 비휘발성 메모리에 저장되므로, 재부팅 후에도 마지막 설정이 그대로 유지됩니다. 색상은 [전체 저장]을 누른 시점에 저장됩니다.

색을 바꿨는데 화면이 안 바뀝니다

다음 순서로 확인하세요.

  • 행별 [설정]을 눌렀는지 — 라벨이 "저장 대기"로 바뀌어야 함.
  • 하단의 [전체 저장]까지 눌렀는지 — 이걸 안 누르면 보드로 값이 안 갑니다.
  • 연결 램프가 연결됨(녹색)인지 — 미연결/오류 상태에서는 저장이 무시됩니다.

기본값(빨/녹/노/회색)으로 되돌리고 싶습니다

[기본값 복원] 버튼으로 입력칸을 출고 기본값으로 채운 뒤, [전체 저장]을 눌러 보드에 반영하세요.
