# LDDisplay 안드로이드 앱

CH340 시리얼 통신용 안드로이드 앱. USB OTG 케이블로 보드와 연결해 사용.

## 설치

### 옵션 A. 직접 다운로드
1. [LDDisplay.apk](LDDisplay.apk) 파일을 폰에서 다운로드
2. 알림에서 파일 열기 → "출처를 알 수 없는 앱 설치" 허용
3. 설치 완료

### 옵션 B. 다운로드 링크 (직접 다운로드용)
```
https://github.com/DainGisul/dain_dist/raw/main/forAndroid/LDDisplay.apk
```

## 요구 사항

- Android 8.0 이상 (권장)
- **USB OTG 지원 폰** (대부분 Android 폰이 지원)
- USB OTG 케이블 (C-to-C 또는 A-to-C)

## 지원 시리얼 칩

- CH340 / CH341
- CH343 / CH9102
- (기타 usb-serial-for-android 지원 칩)
